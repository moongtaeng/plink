PLink는 무엇인가요?

웹서버 접속환경을 한결 편하게 해줍니다. 이제 해볼까요?

서버 접속, 이젠 Proxy로!

웹개발환경 접속할 때 불편한 Hosts 파일 수정은 이제 그만하셔도 됩니다. Proxy 설정으로 한결 가볍게 접속하세요.

Fiddler 확장 모듈

Fiddler 를 사용해서 만든 .Net 확장 모듈입니다. .Net 을 하실 수 있다면 좀 더 재미난 것들을 많이 만드실 수 있습니다.

쉬운 개발서버 접속

HOST, URL, REAL, PATTERN 의 4가지 타입을 활용해서 서버접속을 좀 더 쉽게 만들어줍니다. 파일서비스의 경우 웹서버가 없어도 됩니다.
